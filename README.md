# Diabetes Prediction using Machine Learning

## Project Overview
This project builds a machine learning model to predict whether a person is likely to have diabetes based on several medical features.

## Dataset
The dataset includes medical attributes such as:
- Glucose
- Blood Pressure
- BMI
- Age
- Insulin
- Skin Thickness

## Data Preprocessing
Several preprocessing steps were applied:
- Handling missing values
- Detecting and treating outliers
- Feature analysis

## Exploratory Data Analysis
Different visualizations were created to understand relationships between the features.

## Machine Learning Models
The following models were trained and evaluated:
- Logistic Regression
- Random Forest
- XGBoost

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- xgboost

## Results
The models were evaluated and compared to select the best model for predicting diabetes
